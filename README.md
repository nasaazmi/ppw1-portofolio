## Format Commit

Repository ini menggunakan format **Conventional Commits:**

| Prefix | Digunakan untuk |
|--------|----------------|
| `feat:` | Menambah tugas atau fitur baru |
| `fix:` | Memperbaiki kesalahan |
| `docs:` | Perubahan dokumentasi |
| `style:` | Perubahan tampilan atau styling |
| `refactor:` | Perapian kode |
