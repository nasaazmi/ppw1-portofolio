Tugas Pertemuan 2

Membuat halaman web menggunakan struktur semantic HTML yang terdiri dari header, nav, main, section, article, aside, dan footer. Ditambah dengan tabel data diri berisi nama dan usia.


Catatan Belajar

Hal baru yang saya pelajari di pertemuan ini adalah cara membuat tabel, menggunakan link antar halaman, dan memahami fungsi tag semantic HTML agar struktur website lebih rapi dan terorganisir.


Materi yang Dipelajari

  • Tag Tabel HTML
  • Tag Link / Anchor
  • Semantic HTML
  • Inline Style / Warna
